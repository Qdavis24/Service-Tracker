Service Tracker
Service Tracker is a personal web application built with Python and Flask that helps users keep track of their vehicle service records. This project allows vehicle owners to log, view, and manage their service history online, ensuring they stay on top of regular maintenance and repairs.

Features
Add Service Records: Users can log vehicle service records, including details like date, type of service, and mileage.
View Service History: Users can view a comprehensive list of past services for each vehicle.
Edit and Update Records: Users can modify or update existing records to ensure accuracy.
User Authentication: Secure login and account management, ensuring users' service data remains private.
Responsive Design: Mobile-friendly interface that adapts to different screen sizes.
Tech Stack
Backend: Python, Flask
Frontend: HTML, CSS (using Tailwind CSS)
Database: PostgreSQL
Libraries and Tools:
SQLAlchemy for database interactions
Flask-Login for user authentication
Psycopg2 for PostgreSQL connections

Usage
Register for an account or log in to start adding your vehicle service records.
Add new service records through the user-friendly interface.
View, edit, or delete records at any time to maintain an accurate service history for your vehicle(s).
